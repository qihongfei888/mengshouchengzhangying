# 童心宠伴班级管理系统

## 项目简介

童心宠伴是一个面向班级管理的系统，主要功能包括：
- 学生管理
- 班级管理
- 积分管理
- 奖励系统
- 宠物养成
- 数据同步

## 技术栈

- 前端：HTML, CSS, JavaScript
- 本地存储：localStorage, IndexedDB
- 云端存储：Supabase
- 数据导入：XLSX

## 部署步骤

### 1. 本地开发

1. 克隆项目到本地
2. 打开 `index.html` 文件即可运行

### 2. GitHub Pages 部署

1. 登录 GitHub 账号
2. 创建一个新的仓库
3. 将项目文件上传到仓库
4. 在仓库设置中开启 GitHub Pages
5. 选择 `main` 分支作为源
6. 访问生成的 GitHub Pages 网址

### 3. 配置说明

- **Supabase 配置**：在 `app.js` 文件中修改 Supabase 连接信息
- **同步设置**：默认每5分钟同步一次，累积10个变更触发同步
- **存储设置**：优先使用 IndexedDB，localStorage 作为备用

## 功能特性

- **离线支持**：无网络时仍可正常使用
- **多端同步**：支持在不同设备间同步数据
- **数据备份**：自动备份数据到云端
- **响应式设计**：适配不同屏幕尺寸

## 注意事项

- Supabase 免费计划有 API 请求限制（50,000次/月）
- 建议在生产环境中使用付费计划以获得更好的性能
- 定期清理过期数据以保持存储空间
